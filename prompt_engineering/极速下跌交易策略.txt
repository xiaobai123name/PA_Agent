【文件定位】在已确认极速下跌中，给出**顺势做空**方案（仅 SPS 反弹后入场）。禁止 SCS、禁止追高潮、禁止逆势做多。

⚠️ **PA Agent 硬约束**：禁止 SCS/市价追尖峰、高潮后禁追空、禁止逆势、禁止仓位管理、不依赖成交量。

【核心知识】

一、尖峰期间允许的策略（仅 SPS）

**废止**：策略1 市价追跌、策略2 止损追跌破（原 SCS）。

⚠️ 禁止的是**追价**（市价追跌、对尖峰棒/高潮棒本身挂追跌破单）。反弹后的**信号棒突破触发**（挂单等待、非追价）不在禁止之列，对应 `entry_intent=breakout`、`order_type=突破单`（见二元决策 §9.0B：spike 周期突破单优先）。

策略A：反弹入场（SPS）——尖峰暂停或反弹后：反弹中**信号棒收盘 → 其低点下方 1 跳挂突破单（优先）**；或在反弹位挂限价做空。

策略B：旗形反弹（尖峰旗形顺向突破）——顺向跌破旗形，非追高潮棒；挂旗形下边界下方 1 跳突破单。

二、尖峰到反弹/通道转换

尖峰后的三种发展路径：
1. 趋势恢复为下跌通道（概率最高，约60%）
2. 进入交易区间（概率中等，约30%）
3. 趋势反转（概率最低，约10%，但下跌中反转概率略高于上涨）

交易策略选择：

路径1：下跌通道恢复（高概率）
- 识别特征：反弹幅度<50%，通道倾斜向下，反弹呈波浪形
- 策略：在通道上轨（下降趋势线）附近做空
- 止损位：顺势信号棒/反弹高点上方1跳
- 目标位：通道下轨或测量通道高度结构候选

路径2：进入交易区间（中等概率）
- 识别特征：反弹幅度>61.8%，出现盘整状态
- 策略：仅 direction 仍偏空时在区间上沿评估做空；**禁止双边刮头皮**

路径3：趋势反转（低概率）
- 识别特征：出现卖出高潮，后续出现明显反转K线
- **禁止做多**；仅 `reversal_attempt` / `mtr` 诊断 + `watch_points`
- **不输出**逆原趋势三价；`order_type=不下单`，`terminal=wait`

三、~~追空（SCS）~~（已废止）

**禁止**尖峰追空、SCS。`climax_risk≠none` 时 wait 或等 SPS。

四、卖出高潮后的交易策略

卖出高潮（统一规则，见市场诊断框架）：
- 强趋势序列 + 衰竭信号；**非**「2–5 根即高潮」

卖出高潮后：**禁止追空**；仅 wait 或 SPS 反弹后做空（顺主方向）。

十一、~~尖峰中做多~~（已废止）

下跌尖峰/高潮后**禁止做多**；仅诊断 + wait。

五、下跌趋势日和开盘起下跌趋势日的交易

开盘起下跌趋势日的定义：
- 价格在开盘后的前几根棒线内形成当天的最高点
- 开盘区间小于当天区间的25%
- 开盘区间被向下突破后，价格朝突破方向持续运动
- 与上涨趋势日类似，约20%的交易日属于开盘起下跌趋势日

开盘起下跌趋势日的交易步骤：
1. 识别开盘区间：开盘后前3-6根K线形成的价格范围
2. 等待跌破：价格跌破开盘区间下边界
3. 确认信号：跌破后至少有一根空头趋势棒确认
4. 入场：在跌破后的反弹中入场做空
5. 止损：放在开盘区间上侧边界外

下跌趋势日的特征：
- 开盘后迅速形成当天的最高点
- 反弹幅度极小（<30%日均波幅）
- 每次反弹都是好的做空入场点
- 价格可能收在接近最低点的位置

下跌趋势日的交易规则：
- 如果开盘低点在第一小时被突破，趋势日失败
- 如果开盘高点在第一小时被突破，趋势日失败
- 趋势日确立后，反弹是好的做空入场点
- 下跌趋势日通常在欧美交易时段交界处最易发生

六、小幅反弹趋势日（Trend Day with Small Pullback）的做空交易

小幅反弹趋势日的特征：
- 最强的下跌趋势日类型，一个月可能只出现1-2次
- 当天的反弹幅度仅为日均波幅的10%-30%
- 所有的反弹都是好的做空入场点
- 空头趋势棒和反弹棒交替出现，呈"台阶式"下跌

小幅反弹趋势日的识别：
- 开盘后迅速形成当天的最高点
- 价格快速下跌后出现小幅反弹
- 反弹棒明显小于趋势棒（约30%-50%）
- 反弹时间很短（1-3根棒线）
- 恢复下跌后继续创新低

小幅反弹趋势日的做空策略：
1. 首次反弹入场：在第一次明显反弹时做空
2. 每次反弹都是机会：价格每次反弹到下降趋势线都是做空机会
3. 止损设置：放在最近反弹高点上方
4. 目标：TP1/TP2 取结构位；须 §10.3 通过
5. ~~加仓~~【背景知识】本 Agent 只输出单次方案，禁止写加仓

七、止损设置原则

止损类型1：尖峰结构止损
- 默认定位：最近反弹K线或最近趋势棒高点上方1跳
- 尖峰起点：只有在它本身是本笔交易的结构失效位时才能使用，不能因 RR 选择
- 止损距离过大：保持结构止损，复核或放弃；不得硬缩止损
- 适用：3根以上标准尖峰且结构目标明确时

止损类型2：信号棒过长时不交易
- 定位：标准结构止损过远时保持结构失效位，不在信号棒内部硬设较紧止损
- 处理：等待更小信号或放弃；执行层若调整仓位，必须保留该结构止损
- 原因：人为缩窄止损会把结构失效位改成噪音位，容易被正常波动触发
- 适用：信号棒明显过长、当前结构无法承载风险时

止损类型3：技术位止损
- 定位：将止损放在关键技术位上方/下方
- 技术位：前高、下降趋势线、均线、斐波那契位
- 优点：反映市场结构变化
- 缺点：需要准确识别技术位
- 适用：有一定技术分析基础时

止损设置的安全规则：
- 永远不要将止损放在精确的最高价/最低价
- 至少留出1-2个跳动点的缓冲空间
- 止损一旦设置，不要因为害怕被扫而移动

做空止损的特殊注意事项：
- 恐慌性下跌中，价格可能在止损位附近剧烈波动
- 考虑使用"限价止损"（Stop Limit Order）避免大滑点
- 或者使用更宽的止损 + 更小的头寸来应对波动

八、利润目标设置

目标类型1：测量运动目标（Measured Move）
- 计算方法：从尖峰起点测量到尖峰终点
- 第一目标：测量运动的100%（等距）候选
- 第二目标：测量运动的138.2%或161.8%候选
- 约束：先确定结构失效位与缓冲 stop，再定结构 TP1/TP2；若 RR>1.5 则复核依据，成立则允许，否则拒绝，不能改写 stop
- 优点：有客观的计算依据
- 缺点：可能过于机械

目标类型2：通道高度目标
- 计算方法：测量通道的高度作为目标
- 通道高度 = 通道上轨 - 通道下轨（绝对值）
- 目标位 = 通道下轨 - 通道高度（或倍数）
- 适用：尖峰后进入通道的情况

目标类型3：技术位目标
- 识别前一个重要的支撑位/阻力位
- 将其作为利润目标
- 常见技术位：前低、前高、整数心理位、均线

**PA Agent 单次三价补充**（替代原「分批出场 / 逐步出场 / 连续高潮持仓管理」章节）：

- **TP1** = 近端结构目标；**TP2** = MM/通道对边等更远目标（见文件23）
- 定价顺序：entry → 唯一结构失效位与缓冲 stop → TP1 → TP2 → 计算 RR
- 若 RR>1.5：**保持 TP1/TP2 与结构 stop 不变**，复核依据成立则允许，否则拒绝方案
- 程序要求 RR≥1.0 且 §10.3 交易者方程通过；RR>1.5 需 high_rr_review 通过
- **连续高潮后**：禁追原方向；watch_points 写等待 SPS，不写持仓管理指令
