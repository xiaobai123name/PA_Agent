【文件定位】在已确认极速上涨中，给出**顺势做多**方案（仅 SPS 回撤入场）。禁止 SCS 追尖峰、禁止追高潮、禁止逆势做空。

⚠️ **PA Agent 硬约束**：禁止 SCS/市价追尖峰、高潮后禁追多、禁止逆势、禁止仓位管理、不依赖成交量。

【核心知识】

一、尖峰期间允许的策略（仅 SPS）

**废止**：策略1 市价追尖峰、策略2 止损追突破（原 SCS）——**一律禁止**。

策略A：限价/回撤入场（SPS）
- 尖峰出现暂停或回撤后，在回撤位（38.2%–50% 或前棒低点附近）挂限价或等信号棒做多
- 须 `climax_risk=none` 或回撤已消化高潮；§10.3 通过

策略B：旗形回撤（尖峰旗形顺向突破，顺多）
- 第一次旗形整理后，顺向上突破做多（非追高潮棒）

二、尖峰到通道转换

尖峰后的三种发展路径：
1. 趋势恢复为通道（概率最高，约60%）
2. 进入交易区间（概率中等，约30%）
3. 趋势反转（概率最低，约10%）

交易策略选择：

路径1：通道恢复（高概率）
- 识别特征：回调幅度<50%，通道倾斜向上，回调呈波浪形
- 策略：在通道下轨（趋势线）附近做多
- 止损位：顺势信号棒/回撤低点下方1跳
- 目标位：通道上轨或测量通道高度结构候选

路径2：进入交易区间
- 策略：仅 direction 仍偏多时在区间下沿评估做多；**禁止双边刮头皮**

路径3：可能反转
- **禁止做空**；仅 `reversal_attempt` / `mtr` 诊断 + `watch_points`
- **不输出**逆原趋势三价；`order_type=不下单`，`terminal=wait`

三、~~追单（SCS）~~（已废止）

**禁止**尖峰追单、市价追突破、SCS。`climax_risk≠none` 时一律 wait 或等 SPS。

四、高潮后的交易策略

买进高潮（统一规则，见市场诊断框架）：
- 强趋势序列 + 衰竭信号（长尾/小实体/反向棒）；**非**「2–5 根即高潮」

高潮后禁止的行为：
- 不在高潮后立即做多（买进高潮后追涨）
- 不在高潮后立即设置新的多头止损单
- **禁止**评估做空或任何逆势三价

高潮后的正确策略（单次分析）：
1. 在 watch_points 写等待回调/顺主方向 SPS 二次入场条件
2. 反转结构仅作 `reversal_attempt` 诊断，**不下空单**

五、开盘起趋势日（Opening Range Breakout）的交易

开盘起趋势日的定义：
- 价格在开盘后的前几根棒线内形成当天的最低点/高点
- 开盘区间小于当天区间的25%
- 开盘区间被突破后，价格朝突破方向持续运动
- 约20%的交易日属于开盘起趋势日

开盘起趋势日的交易步骤：
1. 识别开盘区间：开盘后前3-6根K线形成的价格范围
2. 等待突破：价格突破开盘区间边界
3. 确认信号：突破后至少有一根趋势棒确认
4. 入场：在突破后的回调中入场
5. 止损：放在开盘区间另一侧边界外

开盘起趋势日的交易规则：
- 如果开盘高点在第一小时被跌破，趋势日失败
- 如果开盘低点在第一小时被突破，趋势日失败
- 趋势日确立后，回调是好的入场点
- 通常在欧美交易时段交界处最常发生

六、小幅回撤趋势日（Trend Day with Small Pullback）的交易

小幅回撤趋势日的特征：
- 最强的趋势日类型，一个月可能只出现1-2次
- 当天的回撤幅度仅为日均波幅的10%-30%
- 所有的回撤都是好的入场点
- 趋势棒和回调棒交替出现，呈"台阶式"上涨

小幅回撤趋势日的识别：
- 开盘后迅速形成当天的最低点
- 价格快速上涨后出现小幅回调
- 回调棒明显小于趋势棒（约30%-50%）
- 回调时间很短（1-3根棒线）
- 恢复上涨后继续创新高

小幅回撤趋势日的交易策略：
1. 首次回撤入场：在第一次明显回撤时入场
2. 每次回撤都是机会：价格每次回调到趋势线都是做多机会
3. 止损设置：放在最近回调低点下方
4. 目标：结构 TP1/TP2，须 §10.3 通过

七、止损设置原则

止损类型1：尖峰结构止损
- 默认定位：最近回撤K线或最近趋势棒低点下方1跳
- 尖峰起点：只有在它本身是本笔交易的结构失效位时才能使用，不能因 RR 选择
- 止损距离过大：保持结构止损，复核或放弃；不得硬缩止损
- 适用：3根以上标准尖峰且结构目标明确时

止损类型2：信号棒过长时不交易
- 定位：标准结构止损过远时保持结构失效位，不在信号棒内部硬设较紧止损
- 处理：等待更小信号或放弃；执行层若调整仓位，必须保留该结构止损
- 原因：人为缩窄止损会把结构失效位改成噪音位，容易被正常波动触发
- 适用：信号棒明显过长、当前结构无法承载风险时

止损类型3：技术位止损
- 定位：将止损放在关键技术位下方/上方
- 技术位：前低、趋势线、均线、斐波那契位
- 优点：反映市场结构变化
- 缺点：需要准确识别技术位
- 适用：有一定技术分析基础时

止损设置的安全规则：
- 永远不要将止损放在精确的最低价/最高价
- 至少留出1-2个跳动点的缓冲空间
- 止损一旦设置，不要因为害怕被扫而移动

八、利润目标设置

目标类型1：测量运动目标（Measured Move）
- 计算方法：从尖峰起点测量到尖峰终点
- 第一目标：测量运动的100%（等距）候选
- 第二目标：测量运动的138.2%或161.8%候选
- 约束：先确定结构失效位与缓冲 stop，再定结构 TP1/TP2；若 RR>1.5 则复核依据，成立则允许，否则拒绝，不能改写 stop
- 优点：有客观的计算依据
- 缺点：可能过于机械

目标类型2：通道高度目标
- 计算方法：测量通道的高度作为目标
- 通道高度 = 通道上轨 - 通道下轨
- 目标位 = 通道下轨 + 通道高度（或倍数）
- 适用：尖峰后进入通道的情况

目标类型3：技术位目标
- 识别前一个重要的阻力位/支撑位
- 将其作为利润目标
- 常见技术位：前高、前低、整数心理位、均线

**PA Agent 单次三价补充**（替代原「分批出场 / 逐步出场 / 连续高潮持仓管理」章节）：

- **TP1** = 近端结构目标；**TP2** = MM/通道对边等更远目标（见文件23）
- 定价顺序：entry → 唯一结构失效位与缓冲 stop → TP1 → TP2 → 计算 RR
- 若 RR>1.5：**保持 TP1/TP2 与结构 stop 不变**，复核依据成立则允许，否则拒绝方案
- 程序要求 RR≥1.0 且 §10.3 交易者方程通过；RR>1.5 需 high_rr_review 通过
- **连续高潮后**：第二/第三高潮禁追原方向；在 watch_points 写等待回撤/二次入场，**不写**减仓/平仓指令

十一、~~尖峰中做空~~（已废止）

尖峰/高潮后**禁止做空**；仅诊断 + wait。

【实战规则 — PA Agent 单次方案】

规则1：仅评估 SPS 回撤做多；禁止 SCS、禁止高潮后追多。

规则2：止损在结构失效位外；RR 只负责校验，RR>1.5 时复核依据，成立则允许，否则拒绝，不改写止损。

规则3：禁止交易：高潮后追多、假突破追多、无结构止损、多窗口严重冲突。

【例外与陷阱】

陷阱1：假突破后追多 → wait。

陷阱2：~~追单~~ → SCS 已废止，禁止追尖峰。

陷阱3：止损贴噪音 → §10.2 放弃。

陷阱4–7：（略）核心：**不逆势、不追高潮、不写仓位管理**。
- 凭感觉交易，看到上涨就入场
- 没有止损位，没有目标位
- 结果：随机亏损
- 解决：每一笔交易都必须有完整的交易计划
