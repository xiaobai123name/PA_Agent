【文件定位】本文件职责：识别反转、区间边界、突破回踩与趋势回撤中的二次入场机会（如 High2/Low2）。它不是只服务于反转态；凡是第一次信号质量不足、逆势、边界测试或突破回踩，都应优先检查二次入场。

⚠️ **PA Agent 适配**：
1. 只输出单次 entry/stop/TP1/TP2；禁止减仓、仓位比例、持仓后管理。
2. **顺势二次入场**（H2/L2、通道回撤、突破回踩、区间边界顺 direction）→ 可评估三价（§9/§10.3）。
3. **逆势二次入场**（楔形反转、MTR、双重顶底反手、趋势线逆突破）→ **仅诊断** + `watch_points`；**禁止逆势三价**。
4. **三推/末端楔形**出现 → **禁止追顺势**、**禁止逆势反手**（见文件14）。

【核心知识】

一、为什么需要二次入场

1. 市场的基本概率事实
   - 第一次反转尝试多数失败（概率锚点见 `提示词大纲` 简表与 `文件25` §1）
   - 第一个反转信号经常被测试并失效
   - 突破后的首次回撤通常是陷阱
   - 机构交易者利用第一次信号的不确定性进行建仓和洗盘

2. 二次入场的统计学基础
   - 第一次反转信号经常失败，尤其是在强趋势、Always In 或缺少趋势线突破时
   - 第二次反转信号通常更可靠，但不是固定胜率保证
   - 可靠性提升来自更多价格行为确认，而不是形态名称本身
   - 但代价是可能错过部分利润空间

3. 60/40规则（The 60/40 Rule）
   - Al Brooks的核心洞察：60%的交易结果是小盈亏（小赚+小亏），相互抵消
   - 只有约20-30%的交易产生大利润（趋势交易）
   - 约10-20%的交易产生大亏损（止损被打后市场反转）
   - 真正的交易优势不在于胜率，而在于：小盈亏≈0，大利润>>大亏损
   - 二次入场通过提高成功率，减少"小亏损"的数量，从而改善整体盈亏比

4. 二次入场的心理学意义
   - 避免了第一次信号的FOMO（错失恐惧）陷阱
   - 提供了更多的确认信息
   - 给交易者更好的风险回报比
   - 减少了情绪化交易

二、二次入场的类型

1. 楔形反转中的二次入场【Brooks 背景；本 Agent 禁止逆势三价】
   a) 形态：楔形形成→突破→回撤到楔形边界→（Brooks 二次入场背景）
   b) **PA Agent**：仅诊断 `wedge`/`reversal_attempt`；禁止追顺势、禁止逆势三价；监测条件写 watch_points

2. 重要趋势反转（MTR）中的二次入场【Brooks 背景；本 Agent 禁止逆势三价】
   a) MTR定义：主要趋势反转通常需要趋势线被有效突破，然后前极点测试失败
   b) **PA Agent**：`mtr` 仅诊断；禁止产出逆原趋势三价（见文件25）

3. 双重顶/底（Double Top/Bottom）中的二次入场【Brooks 背景；本 Agent 禁止逆势三价】
   a) 双重顶/双重底：两次测试相近极点，第二次失败
   b) **PA Agent**：`double_top_bottom` + `reversal_attempt`；禁止逆势三价（见文件28）
   c) Brooks 背景：第二次测试可靠性高于第一次——仅用于 watch_points 监测，非下单依据
      - 第二次测试确认了该极点的阻力/支撑有效性
      - 双重测试后的反转通常更果断

4. 交易区间边界中的二次入场【PA Agent：仅顺 direction 一侧】
   a) 交易区间上边：第一次触及→回撤→第二次触及→**仅当 direction=bearish 或 AIS** 时评估做空；direction=bullish → wait
   b) 交易区间下边：第一次触及→反弹→第二次触及→**仅当 direction=bullish 或 AIL** 时评估做多；direction=bearish → wait
   c) direction=neutral 且 §2.4 非 Always In → **不下单**（禁止双边边界二次入场）
   d) Brooks 背景：第二次触及区间边界的信号更可靠——仍须 direction 门控
   e) 第一次触及可能是突破尝试，第二次才是真正的区间确认

5. 趋势线突破中的二次入场
   a) 趋势线第一次被突破→价格回到趋势线上方→再次突破→二次入场
   b) 第一次突破经常失败（尤其在窄通道中）
   c) 第二次突破更可能成功
   d) 在长程结构窗口上，趋势线回撤（即第一次突破失败的回撤）本身就是二次入场

三、何时可在第一个信号入场【Brooks 背景；PA Agent 逆势场景不适用】

⚠️ 下列情形在 Brooks 原文中可接受第一次入场；**PA Agent 仅适用于顺 direction / Always In 的顺势二次入场**（H2/L2、突破回踩、通道回撤）。**逆势/MTR/楔形反转/区间逆势 fade → 禁止三价**，仅诊断。

虽然二次入场更可靠，以下 Brooks 背景在**顺势**场景仍可参考：

1. 反转架构异常强时【逆势 → 仅诊断，禁止下单】
   - 超大实体的反转棒（实体长度是近期平均的2倍以上）
   - 反转伴随缺口（Gap）
   - 反转棒是强势外包棒（Strong Outside Bar）
   - 反转棒收盘远超信号棒极点
   - 多根K线同时发出反转信号（共振）

2. 初始突破很强时【仅当突破方向与 direction 一致；禁止 SCS 追第一棒】
   - 突破K线实体极大
   - 突破伴随缺口
   - 突破K线收盘接近极点
   - 突破后连续跟随趋势棒
   - 在这些情况下，等待回撤（二次入场）可能导致完全错过交易

3. 交易区间内的楔形反转【逆势 → 仅 `reversal_attempt` 诊断；禁止三价】
   - Brooks 背景：区间内楔形反转止损位常较明确
   - **PA Agent**：仅 watch_points 监测，不下单

4. 关键支撑/阻力位上的反转【逆势 → 仅诊断；顺势回撤可评估 H1/H2】
   - 在极强支撑/阻力位上出现反转信号
   - 支撑/阻力位被多次确认
   - **逆势**反转信号 → 仅诊断；**顺势**回撤至关键位 → 可评估 H1/H2（须与 direction 一致）

5. 单周期多尺度共振【仅当共振方向与 direction 一致时可评估三价】
   - 即时信号窗口的反转信号 + 长程结构窗口的支撑/阻力
   - 多个观察窗口同时出现反转信号
   - 此时第一个信号可能已经有足够确认，但仍需通过信号棒质量、入场棒触发和交易者方程

四、二次入场的止损和目标

1. 止损规则
   a) 基本止损：信号棒极点外1跳动
   b) 楔形二次入场：止损在楔形边界极点外1跳动
   c) MTR二次入场：止损在MTR极点外1跳动
   d) 双重顶/底二次入场：止损在极点外1跳动（或第二次测试极点外1跳动）
   e) 趋势线二次入场：止损在趋势线极点外1跳动
   f) 如果信号棒过长：优先等待更小信号或放弃；§10.2 判过大，禁止用固定跳数替代结构止损

2. 目标规则
   a) 楔形二次入场目标
   - 第一目标：楔形起点
   - 第二目标：楔形高度测量运动
   - 第三目标：更大结构（通道对边、前高/低等）

   b) MTR二次入场目标
   - 第一目标：MTR起点（反转前的极点）
   - 第二目标：等幅测量运动
   - 第三目标：趋势延续目标

   c) 双重顶/底二次入场目标
   - 第一目标：双顶/双底之间的中点
   - 第二目标：区间对边
   - 第三目标：突破后的测量运动

   d) 通用目标原则
   - 二次入场的入场价格通常比第一次入场更差
   - 因此需要更清晰的结构目标和更高胜率来补偿
   - PA Agent：按结构定 TP1/TP2；RR≥1.0 且 §10.3 交易者方程通过即可，高 RR 需 high_rr_review 通过

3. 风险回报比评估
   - 二次入场通常有更多确认，但价格可能更差；不要假设它一定带来更小止损
   - 好的二次入场用更高确认换取更晚入场，必须重新计算止损、目标与交易者方程
   - 理想方案：结构 TP1/TP2 + §10.3 交易者方程通过（无固定 60% 胜率门槛）

五、二次入场的时间周期考量

1. 当前输入周期上的二次入场
   - 最常用的观察窗口
   - 二次入场通常在6-12根K线内出现（6-12根K线）
   - 如果二次入场等待时间超过2小时（24根K线），信号可能已失效

2. 即时信号窗口上的二次入场
   - 更精细的入场点
   - 二次入场可能在数根K线内出现
   - 适合需要精确入场的边界/回撤 setup

3. 长程结构窗口图上的二次入场
   - 等待时间更长（可能数天到数周）
   - 二次入场的信号更可靠
   - 适合波段交易和趋势交易

【分析流程】

步骤1：识别第一个信号
- 观察是否出现反转信号（楔形突破、MTR、双重顶/底等）
- 评估信号强度：强、中、弱
- 如果信号异常强，**且与 direction 一致**，可考虑 H1/L1（须逐条说明为何无需 H2/L2）

步骤2：决定是否等待二次入场
- 信号强度是否足够？（弱信号→等二次入场）
- 是否在关键位置？（关键位置 + 顺 direction → 可评估 H1；逆势 → 仅诊断）
- 风险回报比是否足够？（不够→等二次入场以改善入场价格）
- 当前市场环境？（趋势中等待更谨慎，区间中可更果断）

步骤3：定义二次入场条件
- 楔形反转：二次入场=回撤到楔形边界
- MTR：二次入场=MTR后的第一次回撤低/高点
- 双重顶/底：二次入场=第二次测试极点
- 趋势线突破：二次入场=回撤到趋势线后再次突破

步骤4：等待和监控
- 设置价格警报
- 监控回撤的深度和速度
- 评估回撤过程中的K线形态
- 如果回撤过深，二次入场可能无效

步骤5：执行二次入场
- 确认二次入场信号已触发
- 入场并设置止损
- 确定目标位
- 评估风险回报比

步骤6：方案跟踪（非持仓）
- 若结构失效：写 `invalidation_condition` / watch_points
- 若二次入场失败（止损位被穿过）：`order_type=不下单` 或 wait，重新评估结构

【实战规则】

一、二次入场通用规则

1. 基本原则
   - 大部分反转尝试失败 → 等待二次入场
   - 二次入场通常比第一次入场更可靠，但不能脱离背景机械加分
   - 好的二次入场价格通常与第一次相同或略差；如果明显更优，要警惕陷阱或原结构误判
   - **§9.0P 计划型限价**在边界/EMA 挂更优价属正常，不受本规则约束（见二元决策 §9.0P）
   - 如果二次入场信号模糊，放弃交易

2. 时间规则
   - 二次入场应在第一次信号后的合理时间内出现
   - 当前输入周期：等待不超过2小时
   - 即时信号窗口：等待不超过数根K线
   - 超时未出现二次入场：放弃该交易思路

3. 有效性验证
   - 二次入场点的回撤不能破坏原结构；具体深度由趋势/区间背景决定，不用固定50%机械否决
   - 二次入场点应有明确的K线信号支撑
   - 二次入场前不应出现破坏性的反向K线

二、楔形二次入场规则【Brooks 背景；本 Agent 禁止逆势三价】

- 楔形**回撤**（顺主趋势架构）→ 见文件14，突破后可评估顺势二次入场。
- 楔形**反转**（末端三推）→ 仅诊断 + watch_points；禁止追顺势、禁止逆势三价。

三、MTR二次入场规则【Brooks 背景；本 Agent 禁止逆势三价】

- 见文件25：四组件齐全 → `order_type=不下单`；watch_points 写监测条件。

四、双重顶/底二次入场规则【Brooks 背景；本 Agent 禁止逆势三价】

- 见文件28：第二次极点失败 → 仅诊断；禁止逆势三价。

【例外与陷阱】

1. 二次入场的常见陷阱
   a) 假二次入场
   - 看似二次入场但市场已经进入新阶段
   - 回撤过深，实际上趋势已经恢复
   - 二次入场点没有K线信号支撑

   b) 二次入场延迟太久
   - 等待时间过长，信号时效性降低
   - 市场环境在等待期间已经变化
   - 错过了最佳入场价格

   c) 过度依赖二次入场
   - 不是所有交易都需要等二次入场
   - 某些高概率信号等二次入场反而错过
   - 强趋势中的反转可能只有一个入场窗口

2. 60/40规则的误用
   - 误以为60/40意味着应该频繁交易：错误！60/40指的是结果分布，不是交易策略
   - 误以为小盈亏不重要：小亏损累积也是亏损，减少小亏损数量很重要
   - 忽略大趋势交易的重要性：利润主要来自少数大趋势交易，不要频繁止盈

3. 双重顶/底的特殊陷阱
   - 两个顶/底不在同一价位：需要一定的容差（通常1-2个点）
   - 第二次测试形成更高的顶/更低的底：可能不是双重顶/底而是趋势延续
   - 双重顶/底被突破后回撤：可能形成新的支撑/阻力位

4. 楔形二次入场的特殊陷阱
   - 楔形尚未完成就入场：三推必须全部完成
   - 楔形被突破后直接入场不等回撤：错过更好的入场点
   - 回撤过深导致止损距离过大：此时不应入场

5. 第一次入场的风险
   - 在第一次信号入场时的主要风险：信号失效，价格继续原趋势方向
   - 除非信号异常强或在关键位置，否则应避免第一次入场
   - 如果选择第一次入场，须逐条说明为何无需 H2/L2，且 trade_confidence 通常更低

6. 二次入场失败后的策略
   - 如果二次入场也失败，该方向的反转概率大幅降低
   - 此时**不应再尝试第三次入场**（含末端楔形三推）；仅写 `watch_points` 监测，**禁止**产出三价（与二元决策 §7.4 / §9.6 / §14 一致）
   - 重新评估市场结构，可能需要等待完全不同的交易设置
