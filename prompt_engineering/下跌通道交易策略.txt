【文件定位】在已确认下跌通道中，给出**顺势做空**方案。仅在 direction=bearish（或 AIS）时加载。

⚠️ **PA Agent 硬约束**：**禁止逆势做多**、禁止追高潮、禁止仓位管理、不依赖成交量。

【核心知识】

一、下跌通道交易的核心原则

1. **只做空**（反弹至均线/通道上沿、旗形、假突破失败后做空）。

2. **禁止逆势做多**（通道底、MTR、双底等仅诊断）。

3. 通道类型：窄/常规/宽/微型通道**均只顺势做空**。

二、顺势做空策略详解

策略一：均线回撤卖出（Moving Average Pullback Short）—— Low 1 / Low 2

这是下跌通道中最可靠的顺势做空入场模式。

⚠️ Low 1/Low 2 的精确定义以「文件16-K线信号识别.txt」与「文件19-H1H2-L1L2计数.txt」为准，本文件中的描述仅为交易应用层面的补充说明，不得修改其核心语义。

Low 1 模式：
- 价格从均线（通常为20周期EMA）向下运行后，首次反弹至均线附近
- 反弹过程中形成至少一根空头信号棒（阴线或反转K线）
- 在信号棒低点下方1跳动处设置止损卖出单（Stop Order）
- 或在信号棒高点上方1跳动处设置保护性止损

Low 2 模式：
- Low 1之后，价格再次下跌，然后第二次反弹至均线
- Low 2的反弹幅度通常比Low 1更大，信号也更清晰
- 入场方式和止损设置与Low 1相同
- Low 2在宽通道中比Low 1更常见
- Low 2的可靠性通常高于Low 1，因为经过两次回撤确认后趋势更明确

关键要点：
- 均线（EMA 20）是动态阻力位，在下跌通道中反复发挥作用
- 反弹至均线时的K线形态至关重要：长上影线、阴线吞没都是强信号
- 如果反弹幅度过大（突破前一个LH），则Low模式失效，不应入场
- Low 3（第三次回撤）→ 标 `wedge` / 收缩台阶，**不得**当作 Low 2 追单；仅 wait 或顺主方向 SPS（见文件14、文件19）

策略二：旗形突破卖出（Flag Breakdown Short）

旗形是下跌通道内最典型的持续形态。

识别空头旗形：
- 一段快速下跌后，出现3~10根K线的横盘或小幅反弹
- 旗形的上下边界清晰，类似于一个小型交易区间
- 旗形通常略微向上倾斜（即使在下跌通道中，旗形反弹是正常的）

入场方式：
- 止损单入场：在旗形下边界（支撑位）下方1跳动处设置卖出止损单
- 限价单入场：如果旗形反弹较深，可在旗形上边界（阻力位）附近用限价单卖出

旗形突破的可靠性：
- 旗形越紧凑，突破后的推进越有力
- 旗形持续时间在5~15根K线时效果最佳
- 突破时（不依赖成交量）

策略三：趋势线反弹卖出（Trend Line Bounce Short）

在下跌通道中，价格反弹至空头趋势线后回落是最经典的顺势做空机会。

识别趋势线反弹：
- 价格从通道下边界反弹，接近趋势线
- 在趋势线附近出现拒绝信号：长上影线、阴线吞没、流星线
- 后续K线开始下跌

入场方式：
- 在拒绝信号K线低点下方1跳动处设置卖出止损单
- 或在趋势线被触碰后的第一根阴线收盘时市价卖出

三、逆势结构仅诊断（本 Agent **禁止逆势做多**）

下跌通道在 Brooks 体系中常被视为多头旗形，但 **PA Agent 不在通道底、双底、MTR、趋势线向上突破等位置产出做多三价**。

仅作结构识别时写入：
- `detected_patterns`：`reversal_attempt`、`double_top_bottom`、`mtr`（四组件齐全时）等
- `watch_points`：监测向上突破测试、极点测试失败、通道演变
- `order_type=不下单`；`terminal.outcome=wait`

若 direction 仍 bearish / AIS：继续评估**顺势做空**（Low 1/Low 2、§9.0P、假突破失败后的做空侧二次入场）。方向翻转须有 §2.3 重判依据，且 MTR 仍只诊断、不下逆势单。

止损单入场（Stop Order Entry）：
- 在信号棒的低点下方（卖出）或高点上方（买进）1跳动处下单
- 优点：确保入场，不会错过行情
- 缺点：可能被假突破扫损
- 适用场景：信号清晰、市场动能强、突破交易

限价单入场（Limit Order Entry）：
- 在前一棒的高点附近（卖出）或低点附近（买进）挂限价单
- 优点：获得更好的入场价格，盈亏比更高
- 缺点：可能永远不会成交，错过行情
- 适用场景：通道内反弹交易、窄通道中的顺势交易

Al Brooks的具体建议：
- 卖出限价单设在前一棒高点（或略高于前一棒高点）
- 买进限价单设在前一棒低点（或略低于前一棒低点）
- 在窄下跌通道中，限价单效果优于止损单
- 在宽下跌通道中，两种方式均可（**仅限顺势做空**）

五、微型通道的交易方法

下跌微型通道（2~10棒，几乎无反弹）的交易原则：

1. 第一个反弹是卖出机会
- 微型通道结束后，第一个反弹（通常在第3~5棒内出现）是顺势做空的良机
- 不要在微型通道内部试图做多——程序交易会碾压逆势者

2. 突破微型通道的尝试通常失败
- 如果价格试图向上突破微型通道（即出现高于前高的K线），第一个突破尝试通常会失败
- 交易策略：在突破失败后，顺势做空

3. 特别紧凑的微型通道
- 行为类似于尖峰（Spike）
- 尖峰结束后通常会有两段反弹：第一段至尖峰中点，第二段至尖峰起点
- 交易策略：等待第二段反弹再入场做空，胜率更高

六、窄通道vs宽通道的不同交易方法

窄下跌通道交易方法：
- 只做顺势做空
- 只有计划明确要求等待前一棒高点/反弹位时才使用 pullback 限价；需要确认时使用 breakout，已确认且未偏离时可评估 immediate
- **若有合格信号棒**：可声明 breakout，在信号棒高点上方1跳动触发；最终由 §11 执行解析校验
- 结构止损：最近波段高点或信号棒极点外1跳动
- 利润目标：通道线附近（TP1）+ MM/对侧候选（TP2）

宽下跌通道/台阶交易方法：
- **只做空**（顺 direction / AIS）；禁止通道底逆势做多

- 顺势做空：Low 1/Low 2模式，均线回撤卖出
- 通道底双底/MTR/向上突破 → 仅诊断 + watch_points
- 突破后等回撤确认再入场（突破测试，顺势做空侧）

七、止损设置详解

1. 信号棒止损法（最常用）
- 卖出：止损设在信号棒高点上方1跳动
- 买进：止损设在信号棒低点下方1跳动
- 优点：简单直接
- 缺点：信号棒实体大时止损距离过宽

2. 跟踪止损法（Trailing Stop）【Brooks 背景知识；本 Agent 不输出移止损/跟踪止损指令】
- 初始止损按信号棒法设置
- 当价格向有利方向移动超过1倍通道宽度后，将止损移至盈亏平衡点
- 随后每下跌0.5倍通道宽度，将止损下移一次
- 在窄通道中，跟踪止损应更积极
- 在宽通道中，给予更多空间

3. 波段结构跟踪止损【Brooks 背景；本 Agent 不输出移止损指令】
- 顺势做空：每出现新波段低点，可将结构失效位参考移至最近波段高点上方（仅用于定价 reasoning，非持仓指令）

4. ATR动态止损法
- 卖出风险参考：入场价 + 1.5×ATR
- 买进风险参考：入场价 - 1.5×ATR
- 适用于波动较大的市场；但 ATR 位置不能替代信号棒、波段极点或通道边界等图表结构止损

5. 时间止损法【Brooks 背景知识；本 Agent 用 invalidation_condition 表达结构失效，不输出时间平仓指令】
- 如果入场后5~10根K线内价格没有向有利方向移动，平仓出局
- 适用于窄通道中的限价单交易

八、利润目标

1. 测量运动目标（Measured Move Target）
- 从通道起点到当前位置的高度，投射至突破方向
- 顺势做空目标：从入场点向下投射等于通道高度的距离（作 TP2 候选）

2. 通道高度目标
- 做空目标：当前价格 - 通道高度

3. 缺口测试目标
- 当通道出现向上突破后，市场通常会反弹至通道起点附近
- 这与"缺口测试"（Gap Test）逻辑一致：市场倾向于回到趋势启动点

4. 分阶段平仓策略【Brooks 背景知识；本 Agent 用 TP1+TP2 一次性表达，禁止分批出场】
- 顺势做空：TP1=通道中线候选，TP2=通道线/MM 候选

九、通道突破后的交易策略

1. 顺势突破（向下突破通道线）
- 大多数向下突破会在5根K线内失败
- 这与"下跌通道=多头旗形"一致——向下突破是假突破
- 交易策略：不要追空，等待反弹至突破点再评估
- 如果反弹后再次下跌且突破前低，则可能形成真正的向下扩展

2. 逆势突破（向上突破趋势线）【Brooks 背景；本 Agent 禁止逆势做多】
- 识别向上突破与回撤测试背景，用于 `reversal_attempt` / `watch_points`
- **禁止**产出做多三价；若 direction 仍 bearish → wait

3. 突破失败（Fakeout）
- 价格突破后迅速回到通道内
- Brooks 背景：向下假突破常预示多头旗形延续（通道将向上突破）——**PA Agent 禁止逆势做多**
- **PA Agent 交易策略**：在价格回到通道内后，评估**顺主趋势**的二次入场（做空侧：向上假突破失败后做空；向下假突破仅 `reversal_attempt` 诊断）
- 例（direction=bearish）：向上假突破通道上沿失败后 → 评估 High 2 / §9.0P 做空；**禁止**因向下假突破反手做多

十、下跌通道演变为交易区间时的应对

识别演变信号：
- 趋势线被多次刺穿，反弹幅度越来越大
- LL+LH序列被破坏（出现高于前高的波段高点）
- 通道宽度扩大，斜率减小至接近水平
- 在长程结构窗口上，价格进入横盘状态
- 出现收缩台阶后紧接着方向不明的横盘

应对策略：
- 停止使用通道交易策略，切换至震荡区间策略
- **仅顺 direction 一侧**在边界等待；**禁止**双边低买高卖
- 原趋势线可能成为交易区间的中线或支撑位
- 市场通常会反弹至通道起点（缺口测试）
- 注意：从下跌通道演变为交易区间的过程中，往往会出现最强劲的反弹——这是"真空效应"的结果

真空效应（Vacuum Effect）详解：
- 当市场接近通道上边界（趋势线）时，如果空头认为突破即将发生，他们会平仓
- 空头平仓=买入→加速价格向趋势线运动
- 同样，当市场接近通道下边界时，多头平仓=卖出→加速价格向通道线运动
- 真空效应使得通道边界的突破往往比预期更猛烈

【分析流程】

当你在下跌通道中制定交易策略时，请按以下步骤进行：

第一步：评估通道状态
- 当前是窄通道、宽通道还是微型通道？
- 通道处于演变的哪个阶段？
- 是否出现收缩台阶？
- 是否需要重画趋势线？

第二步：确定交易方向
- **只做空**（顺 direction / AIS）
- 通道底、双底、MTR、向上突破 → 仅诊断，不下单

第三步：选择入场模式
- 顺势做空：Low 1/Low 2、旗形突破、趋势线反弹
- 选择止损单入场还是限价单入场

第四步：确定入场价位
- 具体在什么价格入场？
- 如果使用限价单，挂单价格是多少？
- 如果使用止损单，触发价格是多少？

第五步：计算止损
- 止损设在哪里？
- 止损距离是否相对通道宽度/ATR 过大？

第六步：计算利润目标
- TP1（近端）与 TP2（远端，如 MM）；须 §10.3 通过

第七步：制定应变计划
- 结构失效 → `invalidation_condition` / `watch_points`（非持仓操作）
- 通道突破/转换 → 建议重新跑分析

第八步：输出单次方案
- decision 中给出 order_type + 三价；禁止部分平仓单或跟踪止损

【实战规则】

1. 顺势做空规则
- 只在反弹中卖出，不追向下突破
- Low 1失败后出现Low 2，Low 2的胜率通常高于Low 1
- 均线（EMA 20）是敌人也是朋友：反弹至均线附近才考虑卖出
- 窄通道中，反弹至均线/前棒高点附近且出现合格空头信号棒时，**可评估**顺势做空；每笔仍须 §9/§10.3/§14，**禁止**因「每根高点都像机会」而过度给单

2. ~~逆势做多规则~~（已废止）
- 通道底、双底、MTR → `order_type=不下单`；仅 watch_points

3. 止损规则
- 信号棒止损法是最低标准
- 如果止损距离超过通道宽度的1.5倍，放弃这笔交易
- 本 Agent 不输出跟踪止损指令

4. 利润目标规则
- 有下单时 TP1 与 TP2 均必填
- 禁止写移止损到盈亏平衡或分批了结

5. 通道突破规则
- 第一个向下突破尝试通常失败（下跌通道=多头旗形）
- 向上突破趋势线 → 仅诊断 `reversal_attempt`，**禁止做多三价**
- 向下突破失败且回到通道内 → 评估**顺势做空**二次入场，禁止写「加仓」

6. 频率规则
- 连续两笔结构失效 → watch_points 说明需重判，暂停新方案

7. 收缩台阶特殊规则
- 收缩台阶出现后：顺势做空 trade_confidence 下调
- 收缩台阶最后一轮下推：默认不做空，等向上突破或二次入场

【例外与陷阱】

1. 假信号棒陷阱
- 不是每一根看似反转的K线都是有效的信号棒
- 内包线作为信号棒的效果较差
- 信号棒必须伴随后续的确认K线

2. 过度做空陷阱
- 在窄下跌通道中，每一根K线都看似卖出机会，但不要过度交易
- 特别是通道后期，持续做空可能遭遇猛烈反弹
- 最大的亏损往往发生在"趋势明明要结束了我还在做空"

3. 移动止损过早陷阱【背景知识；本 Agent 不管理持仓】

4. 忽视多头旗形含义的陷阱
- 下跌通道 = 多头旗形（Brooks 背景）：最终向上突破概率可能高于向下突破
- **PA Agent**：只评估顺势做空；向上突破仅 `reversal_attempt` 诊断 + watch_points，**禁止做多三价**
- 通道后期追空须降 trade_confidence，警惕猛烈反弹

5. 真空效应误判陷阱
- 真空效应导致价格快速运动至通道边界，可能被误判为趋势加速
- 在通道线附近看到加速下跌时，不要追空——可能是空头陷阱

6. 突破确认延迟陷阱
- 顺势侧：watch_points 或 §9.0P 限价等待结构确认；禁止「小仓位试探加仓」

7. 单周期多尺度冲突陷阱
- 长程背景与近期方向冲突时，以**近期结构**为交易主方向，长程写入 `htf_context` 作风险提示（见市场诊断框架 场景5/6）
- **禁止**因长程看跌而在近期多头尖峰中做空，或反之
