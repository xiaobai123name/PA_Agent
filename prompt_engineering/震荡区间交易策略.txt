【文件定位】本文件职责：在已确认震荡区间中，给出**顺 direction 一侧**边界入场的具体方案。在诊断结果为震荡区间（trading_range）或趋势型交易区间（trending_tr）时加载。

⚠️ **PA Agent 硬约束**：禁止双边刮头皮、禁止逆势、禁止追高潮、禁止仓位管理、不依赖成交量。

⚠️ 【trending_tr vs trading_range】
| 特征 | trading_range | trending_tr |
|---|---|---|
| 方向 | 常 neutral | 有偏多/偏空 |
| 交易策略 | direction=neutral → **wait** | **仅顺 direction 一侧** |
| 逆势 | **禁止** | **禁止** |

当 cycle_position="trending_tr" 时：
1. direction=bullish → 仅区间下边界做多
2. direction=bearish → 仅区间上边界做空
3. **禁止** MTR/楔形反手、禁止逆势边界刮头皮
4. 禁止中部交易、禁止追突破（含高潮棒）

【核心知识】

一、核心策略（仅顺 direction）

**废止**「低买高卖双边刮头皮」作为默认策略。

- direction=bullish：仅下边界/回撤**做多**
- direction=bearish：仅上边界/反弹**做空**
- direction=neutral：**不下单**（wait）
- 中间 1/3：不交易

二、止损单入场策略

止损单（Stop Order）是初学者在区间交易中的首选入场方式。止损单的特点是：等待市场先给出方向信号，然后以止损单的形式入场，止损位置预先设定。

【策略1：区间底部High 2买入】
适用场景：价格在区间下边界附近，且 **direction=bullish**（或 §2.4 AIL）
操作步骤：
1. 确认当前处于交易区间状态
2. 等待价格到达区间下边界
3. 等待出现 High 2（H2）形态：区间下边界附近的第二次做多尝试，通常是两腿回撤后，价格再次突破前一根K线高点；定义以「文件19-H1H2-L1L2计数.txt」为准
4. 在多头信号棒高点上方1跳动处放置买入止损单
5. 止损位置：多头信号棒低点下方1跳动（区间下边界只作结构参考）
6. 止盈目标：区间对侧或结构 TP1；须 §10.3 交易者方程通过

原理：高2是多头开始反攻的信号，第二高点是入场的好位置。

【策略2：区间顶部Low 2卖出】
适用场景：价格在区间上边界附近，且 **direction=bearish**（或 §2.4 AIS）
操作步骤：
1. 确认当前处于交易区间状态
2. 等待价格到达区间上边界
3. 等待出现 Low 2（L2）形态：区间上边界附近的第二次做空尝试，通常是两腿反弹后，价格再次跌破前一根K线低点；定义以「文件19-H1H2-L1L2计数.txt」为准
4. 在第二低点下方1跳动处放置止损卖单
5. 止损位置：空头信号棒高点上方1跳动（区间上边界只作结构参考）
6. 止盈目标：区间对侧或结构 TP1；须 §10.3 交易者方程通过

原理：低2是空头开始反攻的信号，第二低点是入场的好位置。

【策略3：反转棒确认后止损单入场】
操作步骤：
1. 价格到达区间边界
2. 出现反转棒（RB）：与当前趋势方向相反的强K线
3. 等待下一根K线收盘确认（确认棒）
4. 在反转棒极点外1跳动处放置止损单
5. 止损：反转棒极点外1跳动（总止损2跳动）

适用性：这是最保守的入场方式，适合新手和高度不确定的市场。

【策略4：外包棒（OB）止损单入场】
操作步骤：
1. 价格到达区间边界
2. 出现外包棒（OB）：当前K线完全包含前一根K线
3. 等待市场选择方向（下一根K线突破OB的高点或低点）
4. 在OB的高点上方（做多）或OB的低点下方（做空）放置止损单
5. 止损：OB的另一侧1跳动

三、限价单入场策略

⚠️ **PA Agent**：下列限价单仅当与 `direction` / Always In **同向**时评估；`direction=neutral` 且非 AIL/AIS → wait。

【策略1：空头尖峰后顺空侧回撤做多】——**已废止**（逆势）。仅当 direction 仍偏多且为区间下沿 SPS 时，用 §9.0P 评估做多。

【策略2：多头尖峰后顺多侧回撤做空】——**已废止**（逆势）。仅当 direction 仍偏空且为区间上沿 SPS 时，用 §9.0P 评估做空。

【策略3：测量移动目标位限价单】
⚠️ **PA Agent direction 门控**：
- 区间**向上突破**后的 MM 目标位：仅 `direction=bullish` 或 AIL 时，可将该 MM 作 **TP2 候选**（顺突破方向做多 setup）；**禁止**在 MM 目标位挂逆势做空限价
- 区间**向下突破**后的 MM 目标位：仅 `direction=bearish` 或 AIS 时，可将该 MM 作 **TP2 候选**（顺突破方向做空 setup）；**禁止**在 MM 目标位挂逆势做多限价
- `direction=neutral` 且无 AIL/AIS → 不下单

操作（Brooks 背景，须与上列门控一致）：
1. 计算区间前的运动幅度（从起涨点到区间的距离）
2. 有效突破后：MM 目标 = 突破点 ± 区间高度（见文件23）
3. 阶段二将 MM 写入 `take_profit_price_2` 等结构字段，**非**在 MM 磁铁位逆势 fade
4. 须 §10.1–10.3 通过；禁止为凑 RR 在 MM 附近设噪音止损

适用性：测量移动目标是结构磁力位，用于 TP2/ reasoning，不是逆势刮头皮依据。

【限价单 vs 止损单的对比】
限价单优点：可以提前挂单，不需要盯盘；可能获得更好的入场价格
限价单缺点：如果市场直接越过边界，可能错失机会；也可能接到"飞刀"
止损单优点：确认信号后入场，更可靠
止损单缺点：需要盯盘；可能在确认时价格已经移动了一段

建议：初学者使用止损单，有经验后逐步引入限价单。

四、带刺铁丝（Barbwire）的交易规则

Barbwire是交易区间中的最高密度形态，需要特殊的交易规则：

规则1：不在Barbwire中交易
在Barbwire形成过程中，方向完全不可预测。正确的做法是等待Barbwire形成结束，等待价格以尖峰形式突破Barbwire。

规则2：Barbwire突破方向参考长程结构窗口
Barbwire之后的价格运动方向，**可参考**长程结构窗口的趋势方向作为概率加权。
⚠️ **与二元决策 §2.2 仲裁**：若近期已形成尖峰/Always In 且与长程背景冲突，**近期方向为交易主方向**；长程仅作风险提示与 watch_points，不得否决近期顺势突破评估。

规则3：等尖峰突破Barbwire
Barbwire的突破必须是尖峰级别的——至少3~5根同方向连续趋势棒。普通幅度的突破（只有1~2根趋势棒）大概率会被拉回Barbwire。

规则4：Barbwire突破后等回踩测试
Barbwire突破后，通常会有回踩测试（Breakout Test），但不应假设必然发生。等待回踩测试成功后再入场，胜率远高于在突破瞬间追入。

Barbwire交易的具体流程：
1. 识别Barbwire（紧凑横向整理）
2. 等待尖峰突破Barbwire（**交易主方向**以近期结构/Always In 为准；长程仅作参考，见规则2）
3. 等待回踩测试（价格回到突破点附近）
4. 回踩测试成功（价格在突破点获得支撑/阻力）
5. 顺势入场

五、三角形交易规则

【上升三角形交易】（与二元决策 §6.6 一致）
入场规则：
1. 向上突破：仅 direction=bullish 时，等回踩测试成功后评估做多
2. 向下跌破：仅 direction=bearish 时，等反弹测试后评估做空；否则仅诊断

【下降三角形交易】
入场规则：
1. 向下突破：仅 direction=bearish 时，等回踩测试成功后评估做空
2. 向上突破：仅 direction=bullish 时，等回踩测试后评估做多；否则仅诊断

【对称三角形交易】
等待边界收窄到极致
等待尖峰突破
等待回踩测试
顺势入场

【扩张三角形交易】
不交易

六、时间因素：中间时段 + 中间位置 = 不交易

时间因素是区间交易中常被忽视但非常重要的因素。

【观察窗口的时间效应】
- 在日内交易中，某些时段天然具有更高的波动性和方向性（如开盘30分钟）
- 某些时段天然低波动（如中午12点~14点的美国时段）
- 在低波动时段交易区间，边界反弹的概率更低

【"中间时段 + 中间位置"规则】
"中间时段 + 中间位置 = 不交易"是最重要的禁区规则。

含义：
- 中间时段：低流动性、低波动性的时段
- 中间位置：区间中间区域，不是边界附近
- 在这两个条件同时满足时入场，刮头皮的成功率显著降低

具体场景：
- 下午2点~4点（美国午后低波动期）+ 区间中间位置 → 不交易
- 任何时段的区间中间1/3区域 → 不交易

正确做法：只在边界附近 + 高波动时段交易

七、刮头皮 vs 波段的选择

⚠️ **PA Agent**：只输出单次方案；**仅顺 direction 一侧**边界刮头皮；`direction=neutral` → wait。

【刮头皮（Scalp）】
适用场景：顺 direction 的区间边界短期入场（非双边低买高卖）
胜率要求：高（通常需要55%以上）
风险回报比：按结构定 TP1/stop，须 §10.3 通过（RR≥1.0）；RR>1.5 复核结构止损、TP1 与胜率依据，成立则允许，不改写 stop
操作频率：高（多次交易）

【波段交易（Swing / Position）】
定义：持仓时间为数小时到数天，目标利润是区间宽度的80%~100%
适用场景：区间突破后的大波段（但要注意：80%突破失败）
胜率要求：相对低（40%~50%）
风险回报比：结构 TP1/TP2 + §10.3；PA Agent 只输出单次方案，不管理持仓
操作频率：低（等待机会）

【选择原则】
1. 在区间中，默认评估顺 direction 一侧的边界入场
2. 尖峰突破须等回踩测试，**禁止 SCS**；方向须与 direction 一致
3. 本 Agent 不管理多笔挂单或持仓取舍

八、止损和止盈设置

【止损设置标准】
⚠️ 统一原则：止损 = 信号棒极点外1跳动（唯一标准，不再设多套规则）。各场景止损以「二元决策.txt §10.1」止损表为准。以下为区间交易中的具体应用，不是独立的止损体系：

1. 标准止损：信号棒极点外1跳（做多：信号棒低点下方1跳；做空：信号棒高点上方1跳）
2. 止损过大判断：止损超过区间宽度10% → 过大，等更小信号或放弃
3. Barbwire/铁丝网：不使用边界外止损，须等待突破后以突破棒极点外1跳止损，或等至少3~5跳结构缓冲才考虑

【止盈设置方法】
1. 结构位法：TP1 取区间内有结构支撑的目标（对边、前 swing、宽度 50%~80% 等）；TP2 取更远 MM/翻测位
2. 测量移动法：区间前波段的等距目标可作 TP2 或磁力位参考
3. 边界法：到区间另一边边界（作 TP1 或 TP2，按远近分层）
4. **禁止**在 decision 中写分批出场/仓位比例；单次 JSON 只给一套三价

【交易者方程在止损止盈中的应用】
交易者方程：胜率 × 回报 > 败率 × 风险

举例（区间下边界买入）：
- 止损距离：5跳动（信号棒低点下方1跳动）
- 止盈距离：取结构 TP1（如区间宽度 50% 附近）
- 胜率：60%（基于区间边界的高质量反弹概率）
- 败率：40%
- 计算：0.60 × 回报 > 0.40 × 风险 → 通过

结论：刮头皮须 RR≥1.0 且交易者方程通过；RR 超过 1.5 时复核依据，成立则允许，不改写结构 stop，也不人为截断 TP1。

【关键规则：入场前止损定稿】【Brooks 持仓纪律；本 Agent 入场前定 stop，不输出移止损/跟踪止损】
- 入场前 stop 必须基于结构失效位（文件17）
- 禁止为凑方程收紧或放宽结构 stop；RR 只作 §10.3 校验
- 持仓后的止损移动不适用本 Agent；结构失效用 `invalidation_condition` 表达


【分析流程】

当你准备在交易区间中进行交易时，请按以下步骤执行：

步骤1：确认区间状态
- 识别区间边界（上边界和下边界）
- 评估区间密度（详见「震荡区间分析识别.txt」密度评估框架）
  - 高密度（CV < 0.3）：边界清晰、波段均匀 → **仅顺 direction 一侧**边界 §9.0P 或信号棒
  - 低密度（CV > 0.5）：边界模糊、波段不均 → 降低交易频率，只等强信号；**中部不用限价单**；**边界仍可按 §9.0P 评估计划型限价**
  - Barbwire（紧凑度 < 25%）：不交易，等突破
- 低密度区间和 Barbwire 的处理优先于其他判断

步骤2：确定交易方向
- **仅顺 direction 一侧**：direction=bullish → 只评估下边界做多；direction=bearish → 只评估上边界做空；neutral → wait
- 是否有反转信号？（仅作 `reversal_attempt` 诊断，**不下逆势单**）
- 长程结构窗口作风险提示，不否决近期 direction（§2.2）

步骤3：选择入场方式
- 止损单：等待信号确认后入场（保守，适合新手）
- 限价单：提前挂单在边界附近（激进，适合有经验者）

步骤4：计算交易者方程
- 止损距离：___跳动
- 止盈距离：___跳动
- 预期胜率：___%
- 胜率×回报 vs 败率×风险 → 通过/不通过

步骤5：交易者方程（§10.3）
- 止损/TP1/TP2 结构位 → 方程通过/不通过
- 用 trade_confidence 表达方案质量，不写持仓时间或仓位

~~步骤6：持仓管理~~（PA Agent 不适用）
- 本 Agent 不跟踪开仓后 K 线；结构变化请用户重新跑分析
- 若需表达失效：写 `invalidation_condition` 与 `watch_points`


【实战规则】

规则1：只在边界附近交易
区间中间位置没有任何交易价值。在边界附近等待——不交易是一种决策，耐心等待边界机会是专业交易者的标志。

规则2：止损是唯一的保护
区间中的刮头皮是高频率交易，高频率意味着高错误率。没有止损保护，一笔错误就会吞噬多次盈利。每一笔交易都必须有止损。

规则3：不在Barbwire中交易
Barbwire是方向最不明确的状态。在Barbwire中交易等于在黑暗中被蒙眼飞行。等待Barbwire结束，等待尖峰突破，等待回踩测试。

规则4：突破后等测试
"追突破"是区间交易中最常见的亏损模式。正确的做法：突破后不追（**禁止 SCS**），等待回踩测试。若测试成功且突破方向与 direction 一致 → 顺势入场；若测试失败 → wait 或仅写 `breakout_failure` 诊断，**禁止反手逆势三价**。

规则5：刮头皮止盈须满足结构与 §10.3
优先取区间宽度 50% 附近有结构支撑的 TP1；RR>1.5 时复核结构止损、TP1 与胜率依据，RR<1.0 或复核不成立或方程不通过 → 不下单。

规则6：分批了结【Brooks 背景知识；本 Agent 用 TP1+TP2 表达，禁止写 50% 仓位或分批出场】

规则7：中午时段减少交易
美国东部时间中午12点~下午2点是低波动期，区间边界反弹概率降低。此时段倾向 wait 或降低 trade_confidence，**禁止**写降低仓位比例。

规则8：识别区间中的趋势并跟随
如果区间中出现了明确的趋势（虽然整体仍是区间），可以短暂地"跟随趋势"，但目标只是到区间另一边边界，不要把趋势当成真的趋势来追逐。


【例外与陷阱】

陷阱1：在区间中间追单
"我看到价格朝边界移动了，赶紧追进去"——这是最常见的区间交易错误。追入的风险是：价格可能已经到了边界的另一侧，你的止损空间极小，回报空间也极小。

陷阱2：把刮头皮做成波段
"已经赚了这么多，再等等，看看能不能赚更多"——这是贪婪的陷阱。区间中刮头皮赚的是"确定的20%~50%"，变成波段后面临"80%突破失败"的风险。守住纪律，到了目标就了结。

陷阱3：不设止损
"我会看着盘的，不需要止损"——这是对人类弱点的忽视。市场会制造让你不愿意平仓的情况（锁仓恐惧、损失厌恶），止损是唯一的保护。

陷阱4：在低密度区间中频繁交易
低密度区间的边界不可靠，很多"边界"只是价格的偶然波动，不是真正的支撑/阻力。在低密度区间中降低交易频率，等待更高质量的信号。

陷阱5：在扩张三角形中交易
扩张三角形的边界向外扩散，方向极度不明。在这种形态中，边界只是暂时的，价格随时可能朝任何方向爆发。不交易是最佳选择。

例外1：连续两天的低波动区间
如果同一个品种连续两天处于低波动区间，第三天开盘出现尖峰突破的概率增加。这是已知的高概率事件，可以提前准备。

例外2：区间边界与均线重叠
当均线恰好穿过区间边界时，这个位置的支撑/阻力意义大幅增加。这是刮头皮的高概率机会，可以适当提前入场。
